# J Soft Solutions – Website

Sitio web corporativo y portafolio profesional de J Soft Solutions.
Proyecto versión 1.0.0 desarrollado con tecnologías web estándar.

## Objetivo del proyecto
Presentar la marca J Soft Solutions, sus servicios y el perfil profesional del fundador,
priorizando claridad, rendimiento y accesibilidad.

## Stack actual
- HTML5
- CSS3
- JavaScript (Vanilla)

## Alcance versión 1.0.0
- Sitio estático
- Sin backend
- Sin frameworks
- Enfoque en estructura, contenido y diseño base

## Roadmap futuro
- Migración a arquitectura cliente–servidor
- Backend con Node.js y PostgreSQL
- Frontend con React
- Autenticación básica (JWT)
